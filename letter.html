⚙️ Email Setup (One-Time)

This mailer uses EmailJS (free, no backend).
Sign up → Create a service (Gmail) → Create a template → Copy IDs below.

📋 Setup Guide

1
Go to emailjs.com → Sign up free → "Add New Service" → connect Gmail
2
Create an Email Template. Set To Email = {{to_email}}, Subject = {{subject}}, Body = {{html_body}} (HTML)
3
Copy your Service ID, Template ID, and Public Key from the EmailJS dashboard into the fields above
4
Fill in your love letter below and hit Send Love! 💌
